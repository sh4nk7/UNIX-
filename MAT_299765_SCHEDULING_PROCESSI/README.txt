
Istruzioni per l'esecuzione del programma:

All'interno del progetto sono forniti tutti i file di codice sorgente costituenti il programma, in aggiunta a un file eseguibile, chiamato main.exe e ottenuto dalla sua compilazione, e a un file di testo, chiamato processi.txt, contenente un esempio di compilazione delle informazioni riguardanti il round robin e i singoli processi che si desidera vengano gestiti dai vari algoritmi di scheduling.
Le informazioni riguardanti i processi da eseguire e il round robin devono essere fornite all'interno di un file di testo da inserire nella medesima cartella in cui è contenuto il programma eseguibile.
Una volta lanciato il programma sarà quest'ultimo a richiedere l'inserimento da tastiera del filename, compreso di estensione, di tale file.

Le regole di compilazione del file sono le seguenti:
- la prima riga deve contenenere un numero, rappresentante il valore di round robin;
- dalla seconda riga in poi, ciascuna riga contiene le informazioni di un solo processo riguardanti, nell'ordine, il nome del processo, il suo cpu burst e il suo livello di priorità, divise l'una dall'altra unicamente da una virgola;
- il file non deve contenere alcuna riga vuota;
